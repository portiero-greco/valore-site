┌─────────────────────────────────────────────────────────────┐
│  ΜΕΡΟΣ 1: ΠΡΟΕΤΟΙΜΑΣΙΑ (2 λεπτά)                           │
└─────────────────────────────────────────────────────────────┘

☐ 1. Κατέβασε τον κώδικα από Figma Make
     └─ Κλικ "Download" ή "Export"
     └─ Αποθήκευσε το ZIP

☐ 2. Αποσυμπίεσε το ZIP
     └─ Δεξί κλικ → "Extract All" (Windows)
     └─ ή Double-click (Mac)

☐ 3. Άνοιξε τον φάκελο
     └─ Βεβαιώσου ότι βλέπεις αρχεία όπως:
        • App.tsx
        • package.json
        • README.md
        • components/ (φάκελος)


┌─────────────────────────────────────────────────────────────┐
│  ΜΕΡΟΣ 2: VERCEL DEPLOYMENT (3 λεπτά)                      │
└─────────────────────────────────────────────────────────────┘

☐ 1. Πήγαινε στο https://vercel.com

☐ 2. Κάνε Sign Up
     └─ Κλικ "Sign Up" (πάνω δεξιά)
     └─ Επίλεξε:
        • "Continue with GitHub" (προτεινόμενο)
        • ή "Continue with Email"

☐ 3. Ολοκλήρωσε την εγγραφή
     └─ Συμπλήρωσε username/password
     └─ Επιβεβαίωσε email (αν χρειαστεί)

☐ 4. Στο Vercel Dashboard:
     └─ Κλικ "Add New..." (πάνω δεξιά)
     └─ Επίλεξε "Project"

☐ 5. Import το Project
     └─ Σύρε τον ΦΑΚΕΛΟ του project στη σελίδα
        (drag and drop - όχι το ZIP!)
     └─ ή κλικ "Browse" και επίλεξε τον φάκελο

☐ 6. Configure Project (ΑΠΛΑ ΚΛΙΚ DEPLOY!)
     └─ Vercel θα ανιχνεύσει αυτόματα τις ρυθμίσεις
     └─ ΜΗΝ αλλάξεις τίποτα
     └─ Κλικ "Deploy" (μπλε κουμπί)

☐ 7. Περίμενε...
     └─ Θα δεις progress bar
     └─ Παίρνει 1-2 λεπτά
     └─ ☕ Χρόνος για καφέ!

☐ 8. SUCCESS! 🎉
     └─ Θα δεις "Congratulations!" μήνυμα
     └─ Θα έχεις URL όπως:
        https://valore-xxxxx.vercel.app


┌─────────────────────────────────────────────────────────────┐
│  ΜΕΡΟΣ 3: ΕΛΕΓΧΟΣ (5 λεπτά)                                │
└─────────────────────────────────────────────────────────────┘

☐ 1. Κλικ στο URL που σου έδωσε

☐ 2. Έλεγξε στο Desktop:
     └─ Ανοίγει η αρχική σελίδα;
     └─ Φαίνεται το λογότυπο;
     └─ Λειτουργεί το menu;

☐ 3. Δοκίμασε το Language Switcher:
     └─ Κλικ "EN" / "ΕΛ" (πάνω δεξιά)
     └─ Αλλάζουν τα κείμενα;

☐ 4. Δοκίμασε τα Links:
     └─ About
     └─ History
     └─ Insurance → Solutions for Individuals
     └─ Contact

☐ 5. Δοκίμασε σε Κινητό:
     └─ Άνοιξε τον ίδιο URL στο κινητό
     └─ Δουλεύει το hamburger menu;
     └─ Είναι responsive;

☐ 6. Έλεγξε τις Φόρμες:
     └─ Property Insurance Form
     └─ Health Form
     └─ Contact Form
     └─ Υποβάλλονται;


┌─────────────────────────────────────────────────────────────┐
│  ΜΕΡΟΣ 4: ΒΕΛΤΙΩΣΕΙΣ (Προαιρετικό)                         │
└─────────────────────────────────────────────────────────────┘

☐ 1. Άλλαξε το Vercel URL:
     └─ Vercel Dashboard → Project Settings
     └─ Domains → "Edit"
     └─ Άλλαξε σε: valore-assicurativo.vercel.app
     └─ Save

☐ 2. Πρόσθεσε Custom Domain (αν έχεις):
     └─ Vercel → Settings → Domains
     └─ Add → Γράψε: www.valore.com
     └─ Ακολούθησε οδηγίες για DNS

☐ 3. Αντικατάστησε Logos (αργότερα):
     └─ Δες το αρχείο: REPLACE-LOGOS-GUIDE.md
