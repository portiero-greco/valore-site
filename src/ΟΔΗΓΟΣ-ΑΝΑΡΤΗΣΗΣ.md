# 🚀 Απλός Οδηγός Ανάρτησης - Βήμα προς Βήμα

## Επιλογή 1: VERCEL (Πιο Εύκολη Μέθοδος) ⭐

### Βήμα 1: Κατέβασε τον Κώδικα
1. Κλικ στο κουμπί **"Download"** ή **"Export"** στο Figma Make
2. Αποθήκευσε το αρχείο ZIP στον υπολογιστή σου
3. Αποσυμπίεσε το ZIP (unzip)

### Βήμα 2: Δημιούργησε Λογαριασμό στο Vercel
1. Πήγαινε στο **https://vercel.com**
2. Κάνε κλικ **"Sign Up"** (Εγγραφή)
3. Επίλεξε **"Continue with GitHub"** ή **"Continue with Email"**
4. Συμπλήρωσε τα στοιχεία σου

### Βήμα 3: Ανέβασε το Project
1. Μετά το login, κλικ στο **"Add New..."** → **"Project"**
2. Επίλεξε **"Import"** ή σύρε τον φάκελο με το project μέσα στη σελίδα
3. Το Vercel θα ανιχνεύσει αυτόματα ότι είναι React app
4. Κάνε κλικ **"Deploy"** (μην αλλάξεις τίποτα)

### Βήμα 4: Περίμενε
- Το Vercel θα χτίσει (build) τον ιστότοπο
- Θα πάρει περίπου 1-2 λεπτά
- Θα δεις ένα μήνυμα **"Congratulations!"** όταν τελειώσει

### Βήμα 5: Ο Ιστότοπός σου είναι LIVE! 🎉
- Θα σου δώσει ένα URL όπως: **`https://valore-assicurativo.vercel.app`**
- Κλικ στο URL για να δεις τον ιστότοπο
- Μοίρασε το link σε όποιον θέλεις!

### Βήμα 6 (Προαιρετικό): Βάλε Δικό σου Domain
1. Στο Vercel dashboard, πήγαινε στο **Settings → Domains**
2. Κλικ **"Add"**
3. Γράψε το domain σου (π.χ. `www.valore.com`)
4. Ακολούθησε τις οδηγίες για να ρυθμίσεις το DNS

## Επιλογή 2: NETLIFY (Επίσης Πολύ Εύκολη)

### Βήμα 1: Κατέβασε τον Κώδικα
- Όπως παραπάνω

### Βήμα 2: Πήγαινε στο Netlify
1. Ανοιξε **https://netlify.com**
2. Κάνε **Sign Up** (Εγγραφή)
3. Επίλεξε **"Sign up with GitHub"** ή email

### Βήμα 3: Ανέβασμα με Drag & Drop
1. Μετά το login, θα δεις μια περιοχή που λέει **"Want to deploy a new site without connecting to Git?"**
2. **Σύρε όλο τον φάκελο** του project μέσα εκεί (drag & drop)
3. Το Netlify θα αρχίσει αυτόματα το deploy

### Βήμα 4: Περίμενε
- Θα πάρει 1-2 λεπτά
- Θα δεις ένα μήνυμα επιτυχίας

### Βήμα 5: LIVE!
- Θα σου δώσει URL όπως: **`https://valore-123abc.netlify.app`**
- Μπορείς να το αλλάξεις σε **Site Settings → Change site name**

## Επιλογή 3: GITHUB PAGES (Για πιο προχωρημένους)

### Βήμα 1: Δημιούργησε GitHub Account
1. Πήγαινε στο **https://github.com**
2. **Sign up** (Εγγραφή)

### Βήμα 2: Δημιούργησε Repository
1. Κλικ **"New repository"** (πράσινο κουμπί)
2. Δώσε όνομα: **`valore-website`**
3. Επίλεξε **"Public"**
4. Κλικ **"Create repository"**

### Βήμα 3: Ανέβασε τα Αρχεία
1. Κλικ **"uploading an existing file"**
2. Σύρε ΟΛΑ τα αρχεία από τον φάκελο σου
3. Κλικ **"Commit changes"**

### Βήμα 4: Ενεργοποίησε GitHub Pages
1. Πήγαινε στο **Settings** (του repository)
2. Βρες **"Pages"** στο αριστερό μενού
3. Στο **"Source"**, επίλεξε **"main"** branch
4. Κλικ **"Save"**

### Βήμα 5: Περίμενε
- Θα πάρει 2-5 λεπτά
- Reload τη σελίδα

### Βήμα 6: LIVE!
- Θα εμφανιστεί το URL: **`https://your-username.github.io/valore-website`**

## 📱 Τι να Κάνεις Μετά το Deploy

### ✅ Έλεγξε τα Πάντα:
1. Άνοιξε τον ιστότοπο σε:
   - 📱 Κινητό (iPhone & Android αν μπορείς)
   - 💻 Desktop/Laptop
   - 🖥️ Tablet

2. Δοκίμασε:
   - ✅ Το language switcher (EN/EL)
   - ✅ Όλα τα links στο navbar
   - ✅ Τις φόρμες (Property Form, Health Form, Contact)
   - ✅ Ότι όλες οι εικόνες φορτώνουν
   - ✅ Τα κουμπιά "Learn More" και "Get Quote"

### 📧 Ρύθμιση Email (Προαιρετικό)
Οι φόρμες προς το παρόν δείχνουν ένα μήνυμα επιτυχίας, αλλά δεν στέλνουν πραγματικά email.

Για να στέλνουν email, χρειάζεσαι:
- **EmailJS** (δωρεάν έως 200 emails/μήνα)
- **SendGrid** (δωρεάν έως 100 emails/μέρα)
- **Formspree** (δωρεάν έως 50 submissions/μήνα)

Αν θέλεις, μπορώ να σε βοηθήσω να το ρυθμίσεις!

## 🎨 Αντικατάσταση Λογοτύπων (Αργότερα)

Τώρα ο ιστότοπος χρησιμοποιεί placeholder λογότυπα.

### Πότε να τα αλλάξεις:
- Μπορείς να το κάνεις ΟΠΟΤΕ θέλεις
- Δεν επηρεάζει τη λειτουργία του site

### Πώς να τα αλλάξεις:
Δες τα αρχεία:
- **`REPLACE-LOGOS-GUIDE.md`** - Πλήρεις οδηγίες
- **`ASSETS-CHECKLIST.md`** - Λίστα με όλα τα logos

Σε λίγα λόγια:
1. Κατέβασε/Συγκέντρωσε τα πραγματικά λογότυπα
2. Ανέβασέ τα σε image hosting (Cloudinary, imgBB)
3. Άλλαξε τα URLs στο `/lib/assets.ts`
4. Κάνε deploy ξανά

## 🆘 Συχνά Προβλήματα

### "Δεν μπορώ να βρω το κουμπί Download/Export"
- Στο Figma Make, ψάξε για **"Export"** ή **"Download"** κουμπί
- Συνήθως είναι πάνω δεξιά

### "Το Vercel/Netlify δεν αναγνωρίζει το project"
- Βεβαιώσου ότι ανέβασες **ολόκληρο** τον φάκελο
- Όχι μόνο μερικά αρχεία

### "Οι εικόνες δεν φορτώνουν"
- Αυτό είναι φυσιολογικό αν δεν έχεις αντικαταστήσει τα Figma assets
- Για τώρα θα δουν placeholder images
- Μπορείς να τα αλλάξεις αργότερα

### "Θέλω να αλλάξω το domain name"
**Στο Vercel:**
- Project Settings → Domains → Add

**Στο Netlify:**
- Site Settings → Domain management → Add custom domain

### "Πώς κάνω αλλαγές μετά το deploy?"
1. Κατέβασε ξανά τον κώδικα
2. Κάνε τις αλλαγές
3. Ανέβασε ξανά στο Vercel/Netlify
4. Αυτόματο re-deploy!
